NAME

system-config-users - User and Group Management tool

 

SYNOPSIS

system-config-users

 

DESCRIPTION

system-config-users provides a graphical interface for adding and removing users and groups to the system. Once started system-config-users provides a help feature to assist in learning the interface.

 

OPTIONS

None

 

FILES

/usr/bin/system-config-users
/usr/share/doc/system-config-users*

 

Examples

To run the program type:

system-config-users
