NAME

addgnupghome - Create .gnupg home directories  

SYNOPSIS

addgnupghome account_1 account_2...account_n

 

DESCRIPTION

If GnuPG is installed on a system with existing user accounts, it is sometimes required to populate the GnuPG home directory with existing files. Especially a oqtrustlist.txtcq and a keybox with some initial certificates are often desired. This scripts help to do this by copying all files from oq/etc/skel/.gnupgcq to the home directories of the accounts given on the command line. It takes care not to overwrite existing GnuPG home directories.

addgnupghome is invoked by root as: 

addgnupghome account1 account2 ... accountn
