NAME

securetty - add a tty to /etc/securetty  

SYNOPSIS

securetty tty  

DESCRIPTION

securetty safely adds tty to /etc/securetty.  

NOTES

securetty is not normally meant to be run by hand; it is invoked by the upstart job that sets up the system serial console.
