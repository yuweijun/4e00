NAME

Pod::Perldoc::ToTk - let Perldoc use Tk::Pod to render Pod  

SYNOPSIS

  perldoc -o tk Some::Modulename &
 

DESCRIPTION

This is a ``plug-in'' class that allows Perldoc to use Tk::Pod as a formatter class.

You have to have installed Tk::Pod first, or this class won't load.  

SEE ALSO

Tk::Pod, Pod::Perldoc  

AUTHOR

Sean M. Burke "sburke@cpan.org", with significant portions copied from tkpod in the Tk::Pod dist, by Nick Ing-Simmons, Slaven Rezic, et al.
