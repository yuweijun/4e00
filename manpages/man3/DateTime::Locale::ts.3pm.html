NAME

DateTime::Locale::ts  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ts' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Tsonga.  

DATA

This locale inherits from the DateTime::Locale::root locale.

It contains the following data.
