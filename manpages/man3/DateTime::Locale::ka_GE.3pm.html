NAME

DateTime::Locale::ka_GE  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ka_GE' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Georgian Georgia.
