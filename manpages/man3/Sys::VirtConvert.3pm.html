NAME

Sys::VirtConvert - Convert a virtual guest to use KVM  

DESCRIPTION

Modules under Sys::VirtConvert are used by the virt-v2v(1) and virt-p2v tools. See the virt-v2v documentation for a more detailed description.

The Sys::VirtConvert module provides package information for virt-v2v.  

VARIABLES

Version number.
