NAME

DateTime::Locale::as_IN  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'as_IN' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Assamese India.  

DATA
