NAME

DateTime::Locale::ga  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ga' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Irish.  

DATA

This locale inherits from the DateTime::Locale::root locale.

It contains the following data.
