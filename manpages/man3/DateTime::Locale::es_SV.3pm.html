NAME

DateTime::Locale::es_SV  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'es_SV' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Spanish El Salvador.  

DATA

This locale inherits from the DateTime::Locale::es locale.

It contains the following data.
