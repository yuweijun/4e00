NAME

DateTime::Locale::aa_DJ  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'aa_DJ' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Afar Djibouti.  

DATA

This locale inherits from the DateTime::Locale::aa locale.

It contains the following data.
