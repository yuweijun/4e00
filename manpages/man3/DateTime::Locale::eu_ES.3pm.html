NAME

DateTime::Locale::eu_ES  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'eu_ES' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Basque Spain.  

DATA

This locale inherits from the DateTime::Locale::eu locale.

It contains the following data.
