NAME

DateTime::Locale::ln_CD  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ln_CD' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Lingala Congo - Kinshasa.  

DATA

This locale inherits from the DateTime::Locale::ln locale.

It contains the following data.
