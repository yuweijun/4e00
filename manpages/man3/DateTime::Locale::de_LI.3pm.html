NAME

DateTime::Locale::de_LI  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'de_LI' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for German Liechtenstein.  

DATA

This locale inherits from the DateTime::Locale::de locale.

It contains the following data.
