NAME

DateTime::Locale::zh_Hans_CN  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'zh_Hans_CN' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Chinese Simplified Han China.
