NAME

DateTime::Locale::zh_MO  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'zh_MO' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Chinese Macau SAR China.
