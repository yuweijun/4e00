NAME

DateTime::Locale::fur_IT  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'fur_IT' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Friulian Italy.  

DATA

This locale inherits from the DateTime::Locale::fur locale.

It contains the following data.
