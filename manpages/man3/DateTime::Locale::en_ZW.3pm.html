NAME

DateTime::Locale::en_ZW  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'en_ZW' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for English Zimbabwe.  

DATA

This locale inherits from the DateTime::Locale::en locale.
