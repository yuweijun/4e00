NAME

oddjob.conf

 

DESCRIPTION

The /etc/dbus-1/system.d/oddjob.conf configuration file specifies D-Bus-level access controls for the com.redhat.oddjob service, in particular which users can invoke methods provided by the service. For more information, please see the D-Bus documentation.
