NAME

sos.conf - sosreport configuration  

DESCRIPTION

sosreport uses a configuration file at /etc/sos.conf.  

PARAMETERS

There are 3 sections of configuration in the sosreport configuration file: general, plugins, and tunables.  

[general] OPTIONS

ftp_upload_url Default ftp server to send reports. gpg_keyring Default gpgkey. gpg_recipient GPG recipient smtp_server Mail server  

[plugins] OPTIONS

disable Comma separated list of plugins to disable.  

[tunables] OPTIONS

(plugin, option) Alter available options for defined plugin.
