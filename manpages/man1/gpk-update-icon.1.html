NAME

gpk-update-icon - GNOME PackageKit Update Icon  

SYNOPSIS

gpk-update-icon [ --verbose ]  

DESCRIPTION

This manual page documents briefly the gpk-update-icon command.

gpk-update-icon allows you to show when updates are available in your session.
