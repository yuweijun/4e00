NAME

sclient - sample Kerberos version 5 client  

SYNOPSIS

sclient remotehost
 

DESCRIPTION

sclient will contact a sample server (sserver(8)) and authenticate to it using Kerberos version 5 tickets, then display the server's response.  

SEE ALSO

kinit(1), sserver(8)
