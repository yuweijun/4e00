NAME

gpk-install-mime-type - GNOME PackageKit MimeType installer  

SYNOPSIS

gpk-install-mime-type [ --verbose ] [ mime-type ]  

DESCRIPTION

This manual page documents briefly the gpk-install-mime-type command.

gpk-install-mime-type allows you to install a package that provides a MimeType.
