NAME

pwdx - report current working directory of a process

 

SYNOPSIS

pwdx pids...
pwdx -V

 

DESCRIPTION

The pwdx command reports the current working directory of a process or processes.

 

GENERAL OPTIONS

-Vshow versionDisplays version of program.

 

SEE ALSO

ps(1) pgrep(1)

 

STANDARDS

No standards apply, but pwdx looks an awful lot like a SunOS command.
