NAME

cgclear - unload the cgroup filesystem

 

SYNOPSIS

cgclear

 

DESCRIPTION

This command moves all the tasks inside various cgroups to the root cgroup, deletes all the cgroups and finally unmounts the cgroup filesystem from the system.
