NAME

readmult - a multitrack wrapper for cdda2wav

 

SYNOPSIS

readmult <track a> <title a> <track b> <title b> ...

 

DESCRIPTION

readmult allows simultaneous extraction and naming of multiple tracks with cdda2wav

 

SEE ALSO

cdda2wav(1)
