NAME

sndfile-info - display information about a sound file  

SYNOPSIS

sndfile-info file  

DESCRIPTION

sndfile-info will display basic information about a sound file such as its format, its sample rate, and the number of channels. This information is obtained using libsndfile (http://www.mega-nerd.com/libsndfile/).
