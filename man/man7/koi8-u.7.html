NAME

koi8-u - Ukrainian Net Character Set encoded in octal, decimal, and hexadecimal  

DESCRIPTION

KOI8-U (KOI8 Ukrainian, described in RFC 2310) is the de-facto standard character set for encoding Ukrainian texts. KOI8-U is compatible with KOI8-R (RFC 1489) for all Russian letters, and extends KOI8-R with four Ukrainian letters (in both upper and lower case) in locations that are compliant with ISO-IR-111.
