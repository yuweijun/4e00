NAME

DateTime::Locale::sv_FI  

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'sv_FI' );
  print $dt->month_name();
 

DESCRIPTION

This is the DateTime locale package for Swedish Finland.  

DATA

This locale inherits from the DateTime::Locale::sv locale.

It contains the following data.
