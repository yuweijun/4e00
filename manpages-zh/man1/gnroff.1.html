NAME

nroff - 用 groff 模拟 nroff 命令  

总览 (SYNOPSIS)

nroff [ -h ] [ -i ] [ -mname ] [ -nnum ] [ -olist ] [ -rcn ] [ -Tname ] [ file... ]  

描述 (DESCRIPTION)

nroff 脚本用 groff 模拟 nroff 命令。 -T 选项的参数除了 ascii 和 latin1 以外都将被忽略。 -h 选项同 grotty -h 选项等价。 -i， -n， -m， -o 和 -r 选项同在 troff(1) 中描述的作用是一样的。 另外 nroff 忽略 -e, -q 或者 -s 选项。 -S (安全) 和 -U (不安全) 选项被传给 groff。 默认情况下 -S 被传给 groff。  

参见 (SEE ALSO)

groff(1)， troff(1)， grotty(1)
