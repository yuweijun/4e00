NAME

printf - 格式化并显示数据  

总览 (SYNOPSIS)

printf FORMAT [ARGUMENT]...
printf OPTION  

描述 (DESCRIPTION)

根据 FORMAT 显示 ARGUMENT(s) .

--help
显示 帮助信息, 然后 结束
--version
显示 版本信息, 然后 结束

FORMAT 就象 C 的 printf 函数中 一样 控制 输出数据, 转译 序列 有:

双引号 (double quote)
"\0NNN"
八进制数字 NNN (0 到 3 个 数字) 表示的 字符
\\
反斜杠 (backslash)
"\a"
报警 (BEL)
"\b"
退格 (backspace)
"\c"
不再产生输出
"\f"
走纸 (form feed)
"\n"
换行 (new line)
"\r"
回车 (carriage return)
"\t"
水平 tab
"\v"
垂直 tab
"\xNNN"
十六进制数字 NNN (1 到 3 个 数字) 表示的 字符
%%
单个 %
%b
把 ARGUMENT 视为 翻译了 `\' escape 序列 的 字符串

而且 所有 C 的 格式命令, 如 diouxXfeEgGcs, 首先 结合 ARGUMENT 转换为 正确 的 类型. printf 能够 处理 变量宽度.  

报告 BUGS

发现的 bug 送往 <bug-sh-utils@gnu.org>.  

另见 (SEE ALSO)

printf 的 完整文档 以 Texinfo 手册 的 格式 维护. 如果 正确 安装了 info 和 printf 程序, 使用 命令
info printf

能够 访问到 完整 的 手册.
