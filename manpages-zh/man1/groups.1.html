NAME

groups - 显示用户所在的组

 

总览 (SYNOPSIS)

groups [OPTION]... [USERNAME]...

 

描述 (DESCRIPTION)

--help
显示此帮助, 然后退出
--version
显示版本信息, 然后退出

此命令的效果和 id -Gn 一样. 如果没有指明 USERNAME, 就使用当前进程的.

 

报告错误 (REPORTING BUGS)

发现错误请联系 <sh-utils-bugs@gnu.org>.

 

另见 (SEE ALSO)

groups 的完整文档以 Texinfo 手册的形式维护. 如果正确安装了 info 和 groups 程序, 使用命令
info groups

应该能够显示完整的手册.
