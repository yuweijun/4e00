USLEEP

Section: User Commands (1)
Updated: Red Hat, Inc
Index Return to Main Contents
 

NAME

usleep - 睡眠若干微秒  

总览 (SYNOPSIS)

usleep [number]  

描述 (DESCRIPTION)

usleep 睡眠 指定的 微秒数. 缺省值 是 1.  

选项 (OPTIONS)

--usage 显示 简短 的 使用方法.
--help, -?
显示 帮助信息.
-v, --version
显示 版本信息.
 

BUGS

在 许多 机器 上, 微秒 计数 不够 精确. 精度 只能 到达 -4 或 -5 (注:10的指数).
