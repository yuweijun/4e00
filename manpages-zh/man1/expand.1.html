NAME

expand - 把 tab 符转换为空格符  

总览 (SYNOPSIS)

../src/expand [OPTION]... [FILE]...  

描述 (DESCRIPTION)

把 各文件 FILE 中的 tab 符 转换为 空格符, 然后 写到 标准输出. 如果 没有 指定 FILE, 或者 FILE 是 -, 就从 标准输入 读取 数据.

-i, --initial
不要 转换 非空白符 后面 的 TAB 符
-t, --tabs=NUMBER
设置 tab 的 间隔 为 NUMBER 个 字符, 而 不再是 8
-t, --tabs=LIST
使用 明确的 tab 位置 列表, 每一项 用 逗号 隔开
--help
显示 帮助信息, 然后 结束
--version
显示 版本信息, 然后 结束

可以 用 -NUMBER 或 -LIST 取代 -t NUMBER 或 -t LIST.
