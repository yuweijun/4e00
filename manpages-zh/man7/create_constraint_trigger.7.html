NAME

CREATE CONSTRAINT TRIGGER - 定义一个新的约束触发器  

SYNOPSIS

CREATE CONSTRAINT TRIGGER name
    AFTER events ON
    tablename constraint attributes
    FOR EACH ROW EXECUTE PROCEDURE funcname ( args )
 

DESCRIPTION 描述

CREATE CONSTRAINT TRIGGER 被 CREATE TABLE/ALTER TABLE 内部使用以及被 pg_dump 用于创建那些用于参考完整性的特殊的触发器。  

PARAMETERS 参数

name

 约束触发器的名称。
events

 触发该触发器的事件范围。
tablename

 发生触发器事件的表名称（可能有模式修饰）。
constraint

 实际的约束声明。
attributes

 约束属性。
funcname(args)

 触发器处理所调用的函数。
