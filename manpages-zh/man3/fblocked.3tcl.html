NAME

fblocked - 测试上次输入操作是否耗尽了所有获得的输入  

总览 SYNOPSIS

fblocked channelId

 

描述 DESCRIPTION

如果在 channelId 上的最新的输入操作返回少于其要求的信息，fblocked命令将返回 1，因为所有可获得的输入都被耗尽了。例如，如果在只能获得三个字符用于输入并且没有行结束序列的时候调用 gets， gets将返回一个空串并且随后的 fblocked 调用将返回 1。

 

参见 SEE ALSO

gets(n), open(n), read(n)

 

关键字 KEYWORDS

blocking, nonblocking
