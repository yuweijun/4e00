NAME

pid - 检索进程 id  

总览 SYNOPSIS

pid ?fileId?

 

描述 DESCRIPTION

如果给出了 fileId 参数，则它应当参照用 open 命令建立的一个进程管道。在这种情况下 pid 将返回一个列表，其元素依次是在这个管道中的所有进程的进程标识符。如果 fileId 参照到打开文件不是一个过程管道则这个列表为空。如果未给出 fileId 参数，则 pid 返回当前进程的进程标识符。所有进程标识符都作为十进制字符串返回。

 

参见 SEE ALSO

exec(n), open(n)

 

关键字 KEYWORDS

file, pipeline, process identifier
