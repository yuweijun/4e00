NAME

destroy - 销毁一个或多个窗口  

总览 SYNOPSIS

destroy ?window window ...?

 

描述 DESCRIPTION

这个命令清除用 window 参数给出的窗口，加上所有它们的后代(窗口)。如果清除了窗口 “.”则整个应用也将被销毁。依次销毁这些 window，如果在销毁某个窗口时发生了错误，则这个命令退出而不销毁剩下的窗口。如果 window 不存在不返回错误。

 

关键字 KEYWORDS

application, destroy, window
