NAME

eof - 在通道上检查文件结束状况  

总览 SYNOPSIS

eof channelId

 

描述 DESCRIPTION

如果在 channelId 上的最近的输入操作(例如 gets)期间出现(occur)了文件结束的情况(condition)则返回 1，否则返回0。

 

参见 SEE ALSO

file(n), open(n), close(n), fblocked(n)

 

关键字 KEYWORDS

channel, end of file
