NAME

tk_chooseColor - 弹出用于用户选择颜色的一个对话框。

 

总览 SYNOPSIS

tk_chooseColor ?option value ...?

 

描述 DESCRIPTION

过程 tk_chooseColor 弹出让用户选择一种颜色的一个对话框。下列成对的选项和值是这个命令的可能的命令行参数:

-initialcolor color
指定在颜色对话框弹出的时候在其中显示的颜色。color 的必须是 Tk_GetColor 函数可以接受的形式。
-parent window
使 window 成为颜色对话框逻辑上的父窗口。颜色对话框在其父窗口中显示。
-title titleString
指定一个字符串作为这个对话框的标题来显示。如果未指定这个选项，则显示缺省的标题。

如果用户选择了一个颜色，tk_chooseColor 将以能被 Tk_GetColor 所接受形式返回的颜色的名字。如果用户取消了这个操作，命令将返回空串。  

示例 EXAMPLE


button .b -fg [tk_chooseColor -initialcolor gray -title "Choose color"]
