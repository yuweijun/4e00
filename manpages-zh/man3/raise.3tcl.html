NAME

raise - 改变一个窗口在堆栈次序中的位置  

总览 SYNOPSIS

raise window ?aboveThis?

 

描述 DESCRIPTION

如果省略了 aboveThis 参数则这个命令升高 window，这样它将在堆栈次序上高于它的所有兄弟窗口(它将不被任何兄弟窗口所遮挡(obscure)并将遮挡与它交叠的任何兄弟窗口)。如果指定了 aboveThis 则它必须是一个窗口的路径名字，它要么是 window 的一个兄弟窗口要么是 window.的兄弟窗口的一个后代。在这种情况下，raise 命令将在堆栈次序上把 window 插入到 aboveThis (或 aboveThis 的祖先窗口中是 window 的兄弟窗口的那个窗口)的紧上面；这最终将要么升高要么降低 window。

 

参见 SEE ALSO

关键字 KEYWORDS

obscure, raise, stacking order
