NAME

lappend - 向一个变量上添加列表元素  

总览 SYNOPSIS

lappend varName ?value value value ...?

 

描述 DESCRIPTION

这个命令把由 varName 给出的变量作为一个列表来对待，并把每个 value 参数作为独立的元素添加到这个列表上，元素间用空格分隔。如果 varName 不存在，则用由 value 给出的元素建立这个列表。Lappend 与 append 类似，但它的 values 被作为列表的元素来添加而不是作为原始文本。这个命令提供了建造大列表的一种相对高效的方式。例如，在 $a 很长的时候，“lappend a $b”比“set a [concat $a [list $b]]”更高效。

 

参见 SEE ALSO

list(n), lindex(n), linsert(n), llength(n), lsort(n), lrange(n)

 

关键字 KEYWORDS

append, element, list, variable
