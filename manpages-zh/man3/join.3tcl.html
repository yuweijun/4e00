NAME

join - 通过把列表元素连接在一起来建立一个字符串  

总览 SYNOPSIS

join list ?joinString?

 

描述 DESCRIPTION

list 参数必须是一个有效的 Tcl 列表。这个命令返回由 list的所有元素连接在一起形成的字符串，用joinString 分隔每个毗连的元素对。joinString 参数缺省是一个空格字符。

 

参见 SEE ALSO

list(n), lappend(n)

 

关键字 KEYWORDS

element, join, list, separator
