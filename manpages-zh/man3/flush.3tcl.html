NAME

flush - 为一个通道刷新缓冲的输出。  

总览 SYNOPSIS

flush channelId

 

描述 DESCRIPTION

刷新为 channelId 缓冲的任何输出。ChannelId 必须是一个从以前的 open 或 socket 命令返回的通道标识符，并且它必须是为写而打开的。如果这个通道在阻塞模式下，在所有被缓冲的输出被刷新到通道之前，命令不返回。如果通道在非阻塞下，在所有被缓冲的输出被刷新之前命令可以返回；剩余的将在后台刷新， (速度)与底层文件或设备吸收它的速度相同。

 

参见 SEE ALSO

file(n), open(n), socket(n)
