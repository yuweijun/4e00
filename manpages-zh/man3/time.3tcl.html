NAME

time - 多次执行一个脚本  

总览 SYNOPSIS

time script ?count?

 

描述 DESCRIPTION

这个命令将 count 次调用 Tcl 解释器求值 script (如果未指定 count 则为一次)。它将返回下面这种形式的一个字符串 


503 microseconds per iteration
它指示每次重复操作(iteration)所需时间的平均值，以微秒为单位。测量时间使用流(elapsed)时间，而不是 CPU 时间。

 

参见 SEE ALSO

clock(n)

 

关键字 KEYWORDS

script, time
