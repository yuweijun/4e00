NAME

pkg::create - 为一个给定的包指定构造一个适当的package ifneeded 命令  

总览 SYNOPSIS

::pkg::create -name packageName -version packageVersion ?-load filespec? ... ?-source filespec? ...

 

描述 DESCRIPTION

::pkg::create 是标准 Tcl 库中的一个实用过程。它被用来为一个给定的包指定建立一个适当的 package ifneeded 命令。可用它构造用于 package 机制的一个 pkgIndex.tcl 文件。

 

选项 OPTIONS

支持的参数有:
-name packageName
这个参数指定包的名字。它是必须的。
-version packageVersion
这个参数指定包的版本。它是必须的。
-load filespec
这个参数指定必须用 load 命令装载的一个二进制库。filespec 是有两个元素的一个列表。第一个元素是要装载的文件的名字。第二个可选的元素是装载的这个文件提供命令的一个列表。如果这个过程的列表是空或被省略了，::pkg::create 将为直接装载而设置这个库(参见 pkg_mkIndex)。可以指定任何数目的 -load 参数。
-source filespec
除了指定必须用 source 命令装载的一个 Tcl 库之外，这个参数类似于 -load 参数。可以指定任何数目的 -load 参数。

必须给出至少一个 -load 或 -source 参数。

 

参见 SEE ALSO

package(n)
