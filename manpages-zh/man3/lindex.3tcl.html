NAME

lindex - 在一个列表中检索一个元素  

总览 SYNOPSIS

lindex list index

 

描述 DESCRIPTION

这个命令把 list 作为一个 Tcl 列表来对待并从它返回第 index 个元素(0 参照这个列表的第一个元素)。在提取过程中，lindex 遵守与 Tcl 命令解释器一样的关于花括号、引号和反斜线的规则；但是，变量替换和命令替换不发生。如果 index 是负数或大于等于在 list 中的元素数目，则返回一个空字符串。如果 index 的值是 end，它参照这个列表中最后一个元素，并且 end-整数 参照在列表中倒数的第指定整数个元素。
