NAME

llength - 统计在一个列表中元素的个数  

总览 SYNOPSIS

llength list

 

描述 DESCRIPTION

把 list 作为一个列表对待并返回给出在列表中元素个数的一个十进制数的字符串。

 

参见 SEE ALSO

list(n), lindex(n), lrange(n)

 

关键字 KEYWORDS

element, list, length
