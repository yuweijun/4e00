NAME

lrange - 从一个列表返回一个或多个毗连的元素  

总览 SYNOPSIS

lrange list first last

 

描述 DESCRIPTION

List 必须是一个有效的 Tcl 列表。这个命令将返回由 first 到 last 的元素(包括它们两个)组成的一个新列表。First 或 last 可以是 end (或它的任何缩写) 来参照这个列表的最后的元素。如果 first 小于零，则作为零来对待。如果 last 大于等于在这个列表中元素的数目，则作为 end 来对待。如果 first 大于 last 则返回一个空串。注意: ``lrange list first first'' 不总是与``lindex list first'' 产生相同的结果(尽管对未用花括号包围起来的简单字段是相同的)；但它精确的产生与``list [lindex list first]''相同的结果。
