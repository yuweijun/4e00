ZDRSCL

Section: C Library Functions (3)
Updated: 15 June 2000
Index Return to Main Contents
 

NAME

ZDRSCL - 使用实数量 1/a 乘一个 n 维复向量  

总览 SYNOPSIS

SUBROUTINE ZDRSCL(
N, SA, SX, INCX )

    
INTEGER INCX, N

    
DOUBLE PRECISION SA

    
COMPLEX*16 SX( * )
 

目的 PURPOSE

ZDRSCL 使实数量 1/a 与 n 维复向量相乘。只要最终结果 x/a 不会上溢或下溢，那么这个函数也不会。

 

参数 ARGUMENTS

N (输入) 整型
向量 x 的分量个数
SA (输入) 双精度
数量 a，用来除 x 的每个分量。SA 必须大于0，否则这个子过程将除以零
SX (输入/输出) 16个复数的阵列 (COMPLEX*16 array)，dimension
(1+(N-1)*abs(INCX)) The n-element vector x.
INCX (输入) 整型
向量 SX 连续值间的步进 > 0: SX(1) = X(1) 且 SX(1+(i-1)*INCX) = x(i), 1< i<= n
