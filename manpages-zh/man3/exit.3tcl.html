NAME

exit - 结束应用(程序)  

总览 SYNOPSIS

exit ?returnCode?

 

描述 DESCRIPTION

终止进程，向系统返回 returnCode作为退出状态。如果没指定 returnCode 则它缺省为 0。

 

参见 SEE ALSO

exec(n), tclvars(n)

 

关键字 KEYWORDS

exit, process
