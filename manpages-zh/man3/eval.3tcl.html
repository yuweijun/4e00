NAME

eval - 求一个 Tcl 脚本的值  

总览 SYNOPSIS

eval arg ?arg ...?

 

描述 DESCRIPTION

Eval 接受一个或多个参数，它们一起组成一个包含一个或多个命令的 Tcl 脚本。Eval 用与 concat命令相同的方式连接(concatenate)它所有的参数，把连接后的字符串递归的传递给Tcl 解释器，并返回这次求值的结果(或它产生的任何错误)。

 

关键字 KEYWORDS

concatenate, evaluate, script
