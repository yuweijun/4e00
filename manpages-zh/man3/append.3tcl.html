NAME

append - 向变量追加  

总览 SYNOPSIS

append varName ?value value value ...?

 

描述 DESCRIPTION

向变量 varName 的当前值添加所有的 value 参数。如果 varName 不存在，给她一个等价于所有 value 参数的联接 (concatenation)的值。这个命令提供了一个有效的方式来增殖性地增长(build up)变量。例如，如果 $a 很长，``append a $b'' 比 ``set a $a$b'' 更加有效率。

 

参见 SEE ALSO

concat(n), lappend(n)

 

关键字 KEYWORDS

append, variable
