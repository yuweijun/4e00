NAME

lsearch - 查看一个列表是否包含一个特定的元素  

总览 SYNOPSIS

lsearch ?mode? list pattern

 

描述 DESCRIPTION

这个命令在 list 的元素中进行查找，看它们中是否有一个匹配 pattern。如果有，命令返回第一个匹配的元素的索引。如果没有，命令返回 -1。mode 参数指示这个列表的元素以何种方式与 pattern 相匹配，它必须是下列值中的某一个:

-exact
列表中的元素必须精确的包含与 pattern 相同的字符串。
-glob
Pattern 是一个通配符式样的模式，它匹配使用与 string match 命令相同的规则的每个列表元素。
-regexp
Pattern 被作为一个正则表达式来对待，它匹配使用在 re_syntax 参考页中描述的规则的每个列表元素。

如果省略了 mode 则它缺省为 -glob。

 

关键字 KEYWORDS

list, match, pattern, regular expression, search, string
