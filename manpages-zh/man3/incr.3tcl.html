NAME

incr - 增加一个变量的值  

总览 SYNOPSIS

incr varName ?increment?

 

描述 DESCRIPTION

增加存储在叫 varName 的变量中的值。变量的值必须是一个整数。如果提供了 increment 则向 varName 变量的值加上它的值(必须是个整数)；否则向 varName 加 1。新值被作为一个十进制字符串存储在变量 varName 中并被作为结果返回。

 

参见 SEE ALSO

expr(n)

 

关键字 KEYWORDS

add, increment, variable, value
