NAME

fcloseall - 关闭所有打开的流  

SYNOPSIS 总览

#define _GNU_SOURCE

#include <stdio.h>

int fcloseall(void);  

DESCRIPTION 描述

函数 fcloseall 将所有打开的流与它们底层的文件或功能集合断开。任何缓冲的数据都将首先被写入，使用 fflush(3) 。注意标准流 (stdin，stdout 和 stderr) 也被关闭。  

RETURN VALUE 返回值

这个函数总是返回 0。  

SEE ALSO 参见

fclose(3), close(2), fflush(3), fopen(3), setbuf(3)  

CONFORMING TO 标准参考

函数 fcloseall 是一个 GNU 扩展。
