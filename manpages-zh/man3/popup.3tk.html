NAME

tk_popup - 贴出一个弹出菜单  

总览 SYNOPSIS

tk_popup menu x y ?entry?

 

描述 DESCRIPTION

这个过程在屏幕上给定位置贴出(post)一个菜单并配置 Tk 使这个菜单和它的级联子菜单可以用鼠标或键盘来遍历(traverse)。Menu 是一个菜单组件的名字而 x 和 y 是显示这个菜单的根坐标。如果省略了 entry 或者是它是一个空串，则菜单的左上角被定位在给定点上。否则 entry 给出 menu 中的一个条目的索引，而定位的菜单将使这个条目位于给定点上。

 

关键字 KEYWORDS

menu, popup
