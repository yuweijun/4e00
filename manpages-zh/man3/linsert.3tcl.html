NAME

linsert - 把元素插入到一个列表中  

总览 SYNOPSIS

linsert list index element ?element element ...?

 

描述 DESCRIPTION

这个命令通过在 list 的第 index 个元素的前面插入所有element 参数来生成一个新列表。每个 element 参数变成这个新列表的一个单独的元素。如果 index 小于等于零，则在这个列表的开始处插入新元素。如果 index 的值是 end，或大于等于在这个列表中的元素数目，则在这个列表的尾部添加新元素。end-整数 参照在列表中倒数的第指定整数个元素

 

参见 SEE ALSO

list(n), lappend(n), llength(n)

 

关键字 KEYWORDS

element, insert, list
