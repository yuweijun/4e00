NAME

cursors - 在 Tk 可获得的鼠标光标

移植要点 PORTABILITY ISSUES

Windows
在 Windows 系统上，把下列光标映射成本地光标: 

arrow
center_ptr
crosshair
fleur
ibeam
icon
sb_h_double_arrow
sb_v_double_arrow
watch
xterm
还可获得下列补充的光标: 

no
starting
size
size_ne_sw
size_ns
size_nw_se
size_we
uparrow
wait
可以指定 no 光标来使光标消失。

Macintosh
在 Macintosh 系统上，把下列光标映射成本地光标: 

arrow
cross
crosshair
ibeam
plus
watch
xterm
还可获得下列补充的光标: 

text
cross-hair
