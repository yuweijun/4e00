NAME

break - 终止循环命令  

总览 SYNOPSIS

break

 

描述 DESCRIPTION


 这个命令典型的在一个象 for 或 foreach 或 while 这样的循环命令的(循环)体中调用。它返回一个  TCL_BREAK 代码，这将导致一个 break 例外发生。这个例外将导致当前脚本被退出到(be aborted out to )在最内部(innermost)包含的循环命令，接着这个命令将终止它的执行并正常返回。在一些其他情况下也处理 Break 例外，如 catch 命令、Tk 事件绑定、和过程体的最外部(outermost)的脚本。

 

参见 SEE ALSO

catch(n), continue(n), for(n), foreach(n), while(n)

 

关键字 KEYWORDS

abort, break, loop
