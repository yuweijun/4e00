NAME

iptables-save - 保存 IP Tables  

总览 SYNOPSIS

iptables-save [-c] [-t table]
 

描述 DESCRIPTION

iptables-save 用来将 IP Table 转储为可以简单解析的格式，输出到标准输出 STDOUT。 可以使用 shell 的 I/O 重定向功能来写入文件

-c, --counters
在输出中包含所有报文和字节计数的当前值
-t, --table tablename
限制只输出一个表。如果不指定，会输出所有可能的表
